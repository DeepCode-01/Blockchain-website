FAQ

What is an admin and dashboard template?

It is a Front-end kit with prebuilt UI components for the creation of the interface for the web apps. It contains HTML, CSS, and JavaScript code. You can use it to create an admin console and dashboards.

Why do I need EasyDev?

Using templates saves you time and money because you don’t need to design and code the front-end part from scratch. So it allows you to get up and run your project faster and easier. Also, it contains a few useful ready-to-use integrations and apps. You can use them as an example to build your own or implement them directly into your project. EasyDev has a good structure and code hence fits well for learning purposes.

What skills do I need to have to use EasyDev?

EasyDev is a React-based template. To use it efficiently general knowledge of Front-end development is required. You might need more in-depth skills in React to customize components. Also, keep in mind that the template is not contains the back-end. To make an end-product you need to have knowledge or help with the back-end part.

Do you provide any design frames?

Yes. You will find the link to the Figma frames. You can copy the frames, and customize them as you want in your Figma-project.

Is this landing page a part of the EasyDev template?

Yes. And you are free to use it. However, please remember, if you use it for commercial products you need to buy an Extended license.

What’s the difference between regular and extended licenses?

The regular license can be used for non-commercial end products (access is free and there will be no monthly subscription fee). You need to buy a new single regular license for each product you make or each domain you use if you set up it on multiple domains. An extended license can be used for end products (web service or SaaS) that charge users for access or service (e.g: monthly subscription fee). For more info on Themeforest licenses, you can check http://themeforest.net/licenses/standard?license=regular.
